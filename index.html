Garden Calculator


-> Infinite Expansion mod <-

The Garden Calculator was originally made by Daydun and hosted at https://daydun.com/cookie/garden for the Garden minigame in Cookie Clicker. It was intended to be a way to test the efficiency of your Garden configurations by displaying the probabilities for plants to mutate in a given configuration.

It was upgraded by CursedSliver to include many new features and improvements; mainly for Garden Puzzles, but has also picked up on Daydun's unfinished agenda with the addition of features also helpful to ordinary players.

To report bugs or leave comments/suggestions, go to the github page.
